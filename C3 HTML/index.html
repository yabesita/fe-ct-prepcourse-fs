Esta es la niticia N° 1: Algunas palabras sobre HTML

Un elemento de bloque es aquél que una vez utilizado aparece en la

siguiente línea y ocupa todo el ancho. Elementos de tipo bloque son los

párrafos p, los formularios form, o las cabeceras hx.

Un elemento en línea es aquel que se muestra justo a continuación del

anterior elemento. Estos elementos serían los enlaces a, imágenes, etc.

El lenguaje HTML nos permite agrupar un conjunto de elementos mediante una

agrupaciónen bloque o una agrupación en línea.
